Problem → solution

One link.
One usable file.

A successful transfer was not enough if the result still needed manual codec checks and conversion before it could enter an editing timeline.

01 / Input

Start in Finder

A macOS Quick Action accepts one authorised media link and checks the local tools before starting.

02 / Validation

Resolve and verify

Controlled format fallbacks, output checks and FFprobe validation prevent half-finished results.

03 / Output

Prepare the MP4

Unexpected media is normalised to a predictable H.264, AAC and 1080p editing format.

Technical notes Open details

Failure handling

The workflow checks runtime availability, rejects unsuitable input, tries controlled format variants and confirms that the final file exists and is non-empty.

Responsible scope

Designed only for media I own or have permission to process. This portfolio does not distribute source code, credentials or account data.
