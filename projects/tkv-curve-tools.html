Problem → solution

Shape curves faster.
Stay in the flow.

I needed a focused set of daily animation commands inside Maya Graph Editor, without searching through large menus or opening another toolbox.

01 / Curves

Shape the motion

Pull, Push, Ease, Noise, Wave, Smooth and Rough adjust value and timing without rebuilding curves by hand.

02 / Blending

Control in-betweens

Tweener, Blend to Ease and Blend to Tangent provide direct control over spacing and tangent influence.

03 / Production

Remove repeated steps

Infinity modes, reverse animation, selection tools and key-timing utilities stay in one predictable row.

Technical notes Open details

Selected highlights

Blend to Tangent reacts to the selected handle, Reverse Animation preserves useful timing, and Bake from Last Selected transfers reference key timing.

Context & scope

Familiar interaction ideas were inspired by animBot; the current Python implementation and Graph Editor integration were built for my own workflow and tested in Maya 2025 on macOS.
