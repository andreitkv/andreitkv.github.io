The problem

I needed a focused animation panel, not another oversized toolbox.

I regularly used only a small group of curve and blending operations from larger animation toolsets. The rest added visual noise and pushed frequent commands into menus, hotkeys or separate windows.

I built TKV Curve Tools around familiar animBot-style curve operations and my own daily workflow. The current implementation, Graph Editor integration and production utilities are maintained as one compact personal toolset.

Less searching.
Fewer repeated clicks.
More attention on the motion.

The toolkit

One compact row. Three layers of animation work.

01

  • Bake from Last Selected

Selected highlights

The parts that changed my workflow the most.

01 / Tangents

Blend to Tangent

The slider recognises left and right tangent components separately. Dragging in either direction changes the intended side, so the interaction stays visually connected to the selected handle.

02 / Locomotion

Reverse Animation

A one-click operation for turning forward locomotion into backward motion while preserving the useful timing structure. It replaces a repetitive sequence of curve edits with one controlled action.

03 / Key timing

Bake from Last Selected

Dense animation can be rebuilt using the key timing of the last selected reference control — useful for blocking, cleanup and checking motion in a stepped workflow.

Design decisions

Built for a personal production workflow.

Graph Editor first

The interface stays attached to the place where the work happens instead of opening another floating tool window.

Context-sensitive input

Selection, tangent side and modifier keys change how an operation behaves without multiplying buttons.

Safe iteration

Undo-aware operations and captured curve data keep slider interaction responsive and reversible.

Honest scope

This is a personal tool tested in Maya 2025 on macOS. Compatibility with other environments has not been claimed.

Context & attribution

Several familiar interaction ideas were inspired by animBot, while standard tangent and infinity actions come from Maya. The implementation, compact Graph Editor integration and workflow-specific tools were developed for my own animation process.
