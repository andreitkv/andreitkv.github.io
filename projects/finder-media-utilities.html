The problem

Small repetitive jobs were interrupting larger creative work.

Converting one clip was easy. Repeating the same settings, output names and folder steps across many files was where time and attention disappeared.

I turned the stable versions of those commands into Finder actions. Files can be selected where they already live, processed with predictable settings and returned next to the originals.

The collection

Each utility solves one narrow production task.

01 / GIF

High-quality GIF

Creates a compact preview with a generated colour palette and controlled dithering instead of relying on a basic one-pass conversion.

02 / SEQUENCE

Video to frames

Extracts every frame into a clearly named PNG sequence, ready for review, compositing or image-based workflows.

03 / VIDEO

1080p MP4

Normalises mixed source files into a predictable video format for editing, sharing and portfolio preparation.

04 / TIMING

Slow motion

Produces two practical speed variants while keeping output naming and audio handling consistent.

05 / IMAGES

Batch resize

Turns a folder selection into consistently sized image assets without opening a separate graphics application.

06 / MAYA

Random pose sequence

Selects up to 151 images, unifies their dimensions and names them as a sequence so Maya displays every reference frame correctly.

Shared design

Different outputs, one predictable way of working.

Files stay nearby

Outputs are created beside the selected source instead of disappearing into an unknown temporary folder.

Batch-aware

Actions accept multiple Finder selections and report success or failure for the complete operation.

Explicit settings

Frame rate, size, quality and naming rules live in the script so repeated results remain consistent.

Quiet feedback

System sounds provide a simple completion signal without forcing another application window into the workflow.
